overhang.js

A JQuery plugin to display sleek, instant notifications, confirmations or prompts inside a given element.

Success Message


$("body").overhang({
  type: "success",
  message: "Woohoo! It works!"
});


$("body").overhang({
  type: "error",
  message: "Whoops! Something went wrong!",
  closeConfirm: true
});

Closable Message

Timed Message


$("body").overhang({
  type: "info",
  message: "You won today's sweepstake!",
  duration: 5,
  upper: true
});


$("body").overhang({
  type: "warn",
  message: "A user has reported you!",
  duration: 3
});

Warning Message

Prompt Alert



$("body").overhang({
  type: "prompt",
  message: "What's your name?"
});


$("body").overhang({
  type: "confirm",
  message: "Are you sure?"
});
